Important

Close These Applications Before Exam

Remote access, screen recording, virtualization and chat applications can interfere with SEB's lockdown mode, pose an academic-integrity risk, or consume system resources needed for the exam. Close all of the following before you begin.

Why this matters: SEB automatically detects remote-access and screen-capture tools. If any are running, the exam will not start until they are closed.

System Settings

Disable Background Applications

A step-by-step guide to freeing up system resources before your exam.

1. Task Manager
  1. Press Ctrl + Shift + Esc
  2. Select any non-essential running app
  3. Click End Task
2. Startup Apps
  1. Open Settings > Apps > Startup
  2. Toggle off unnecessary apps
  3. Restart your computer
3. Background Apps
  1. Open Settings > Privacy > Background Apps
  2. Turn off apps not required for the exam
  3. Keep only SEB and system processes running

Permissions

Camera, Microphone & Browser Permissions

SEB needs these permissions to proctor your exam session correctly.

Camera Permission Guide
  • Open SEB
  • Click Allow when prompted for camera
  • Allow the microphone as well
  • Do not click Block
Microphone Permission Guide
  • When SEB requests microphone access, click Allow
  • Speak briefly to confirm the input level indicator moves
  • If muted by default, unmute from system sound settings
  • Never deny microphone access during exam launch
Browser Permission Guide
  • SEB uses its own embedded browser — no separate browser setup is required
  • If prompted, allow SEB to access the microphone and camera at the browser layer too
  • Clear cache if a permission prompt does not appear
Windows

Windows Permission Guide

Enable camera and microphone access at the operating-system level so SEB can use them.

  1. Open Settings
  2. Go to Privacy & Security
  3. Select Camera → toggle Camera access on
  4. Select Microphone → toggle Microphone access on
  5. Ensure "Let apps access your camera/microphone" is enabled
Windows Privacy Path
Settings Privacy & Security Camera Microphone Enable Access
macOS

macOS Permission Guide

macOS requires explicit permission grants for camera, microphone, screen recording and accessibility.

  1. Open System Settings
  2. Go to Privacy & Security
  3. Enable Camera access for Safe Exam Browser
  4. Enable Microphone access for Safe Exam Browser
  5. Enable Screen Recording for Safe Exam Browser
  6. Enable Accessibility for Safe Exam Browser

System Settings Privacy & Security Camera Microphone Screen Recording Accessibility
