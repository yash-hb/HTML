System Check

System & Network Requirements

Please ensure your computer and internet connection meet the specifications below before initiating the exam.

1. Hardware Specifications
Component Minimum Requirement
Processor (CPU)
Dual-core processor or better
RAM
At least 4 GB (8 GB recommended for smooth performance)
Free Disk Space
At least 350MB free storage for installation (1 GB recommended)
Display
Single monitor / single active display only (multi-monitor setups are not allowed by SEB)
Webcam
Working internal or external USB webcam required
Microphone
Working internal or external microphone.
Students are not allowed to wear any headphones, earphones, earbuds, or headsets while taking the examination.
Keyboard & Mouse
Standard physical keyboard and Mouse / Trackpad
Power
Laptop should stay connected to a power source during the exam to avoid shutdown

Keep a mobile hotspot ready as backup connectivity in case main Wi-Fi fails

Download SEB Installer

Download the official Safe Exam Browser installer for your operating system. Do NOT install on mobile devices or tablets.

  • 1Open your browser and go to safeexambrowser.org, or click the Windows / macOS buttons below for a direct download.
  • 2Click Download and choose Windows or macOS installer (approx 335 MB).
  • 3Once the download finishes, double-click the downloaded .exe or .dmg file to start the installer.
  • 4If Windows shows a "User Account Control" pop-up or macOS prompts for permission, click Yes / Allow to let the installer run.
  • 5Follow the on-screen setup wizard: accept the license agreement and click Install, then Finish once complete.
  • 6Whitelist SafeExamBrowser.exe in your antivirus software so it is not blocked or quarantined.
2

Allow Device Permissions

SEB requires camera and microphone access for proctoring. Enable them at the OS level before exam day.

  • 1Windows: Settings → Privacy & Security → Camera → Enable
  • 2Windows: Settings → Privacy & Security → Microphone → Enable
  • 3macOS: System Settings → Privacy → Camera → Allow SEB
  • 4macOS: Also enable Microphone, Screen Recording & Accessibility
  • 5When SEB launches, click Allow on all permission prompts

Launch Your Exam

Double-click the .seb config file to launch SEB directly. Do NOT open SEB manually first.

  • 1Clean Background: Fully quit screen recorders (OBS, Bandicam) and remote desktop platforms (AnyDesk, TeamViewer) to pass SEB security.
  • 2Locate Configuration: Open your Desktop or Downloads folder and select the downloaded file named GTU-Online-MBA-IB-W25.seb.
  • 3Automatic Launch: Double-click the .seb configuration file. Safe Exam Browser will open automatically in secure full-screen mode.
  • 4Allow Access: Click Allow on the prompt asking for camera and microphone permissions. (Crucial for online proctoring).
  • 5Portal Login: Fill in your student registration credentials (username and password) on the GTU login screen.
  • 6Start Exam Session: Click the green Start Exam button in the portal to begin your monitored examination.

Important — Before Exam

Close These Apps First

Remote access, screen recording, and VM software will block SEB from starting. Close all of the following before your exam.

Why this matters:
SEB automatically detects active background remote-access and screen-capture tools. If any are running, the exam will refuse to load.
Troubleshooting: Remote Access Blocks (Chrome Remote Desktop & AnyDesk)

If SEB detects active background remote-access tools (like AnyDesk or Chrome Remote Desktop), follow these steps to stop them:

Step 1: Open Command Prompt as Administrator

Click the Windows Start menu, type cmd, right-click Command Prompt, and select Run as administrator.

5.1 Chrome Remote Desktop
Action Command (click copy icon)
1. Stop the background service
net stop "Chrome Remote Desktop Service"
2. Force-close host process
taskkill /F /IM remoting_host.exe
5.2 AnyDesk
Action Command (click copy icon)
1. Stop the AnyDesk background service
net stop "AnyDesk Service"
2. Force-close AnyDesk application
taskkill /F /IM AnyDesk.exe
Step 2 (Both Tools): Confirm in Task Manager

After running the commands, open Task Manager (Ctrl+Shift+Esc) and confirm that no AnyDesk or Chrome Remote Desktop process is running under the Processes or Services tab. Only then launch SEB for the exam.

Important: If a command returns "service not found", it simply means that particular service is not installed on your computer — this is normal and can be ignored. If a student uses any other remote-access tool (TeamViewer, UltraViewer, etc.), it must be closed the same way via Task Manager before starting SEB.

Step 2

Camera, Microphone & Browser Permissions

SEB needs these permissions to proctor your exam session. Enable them before launching your exam.

Camera Access
  • Open SEB and click Allow when prompted
  • Allow microphone access as well
  • Never click Block
Microphone Access
  • Click Allow when SEB requests mic access
  • Speak briefly to confirm input level moves
  • Unmute from system settings if muted
Browser Permissions
  • SEB uses its own built-in browser — no extra setup needed
  • Allow camera & mic if prompted inside SEB's browser
  • Clear cache if permission prompt doesn't appear

Windows

Windows Permission Guide

Enable camera and microphone at the OS level so SEB can use them.

  1. Open Settings
  2. Go to Privacy & Security
  3. Click Camera → toggle Camera Access ON
  4. Click Microphone → toggle Microphone Access ON
  5. Ensure "Let apps access your camera/microphone" is enabled
Windows Privacy Path
Settings Privacy & Security Camera Microphone Enable Access
macOS

macOS Permission Guide

macOS requires explicit permission grants for camera, microphone, screen recording and accessibility.

  1. Open System Settings
  2. Go to Privacy & Security
  3. Enable Camera for Safe Exam Browser
  4. Enable Microphone for Safe Exam Browser
  5. Enable Screen Recording for Safe Exam Browser
  6. Enable Accessibility for Safe Exam Browser

Exam Day

How to Start Your Exam

Follow these exact steps on the day of your exam, in order. Do not skip any step.

1
Download the .seb file

Click the "Download GTU-Online-MBA-IB-W25.seb" button on this page and save it to your desktop.

2
Double-click the .seb file

Locate GTU-Online-MBA-IB-W25.seb in Downloads/Desktop and double-click it. Do NOT open SEB manually.

3
SEB Opens Automatically

Safe Exam Browser launches in full-screen mode and loads your exam portal directly.

4
Login with Credentials

Enter your GTU username (roll number) and password to authenticate.

5
Allow Camera Permission

When prompted, click Allow to grant camera access. Do not click Block.

6
Allow Microphone Permission

Also allow microphone access when SEB asks. Both camera and mic are required.

7
🎯 Start Exam!

Click the Start Exam button on the portal. Good luck!
